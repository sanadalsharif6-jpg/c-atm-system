# C ATM Banking System

A command-line ATM banking system written in C with file-based account storage.

## Features

- Create accounts
- Login using account number and PIN
- Check balance
- Deposit money
- Withdraw money
- Transfer money between accounts
- Save account data to a file

## How to Compile

```bash
gcc main.c -o atm
```

## How to Run

On Windows PowerShell:

```bash
./atm
```

If PowerShell does not run it, use:

```bash
.\atm.exe
```

## Technologies

- C
- File handling
- Structs
- Functions
